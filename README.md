# SyntaxGym Starter

Local-first code typing trainer for Rust syntax and DSA snippets.

## Run

```bash
pnpm install
pnpm dev
```

Then open the Vite URL.

## What this starter already has

- Vite + React + TypeScript app
- Pure typing engine package
- Rust snippets package
- Rust token-aware mistake analysis
- LocalStorage persistence for custom snippets and session history
- No backend
- No login
- No LeetCode scraping
